5.9. The Settings Menu

Settings->Show Toolbar

Shows or hides the Toolbar

Settings->Show Actions

Shows or hides the Actions

Settings->Show Tools

Shows or hides the Tools

Settings->Show Parameters

Shows or hides the Parameter Bar

Settings->Show Cluster Info

Shows or hides the Cluster information in the Cluster Panel

Settings->Show Status Bar

Shows or hides the Status Bar

Settings->Immediate Update

Triggers immediate redrawing of all views after a change

Settings->Delayed Update

Prevents immediate redrawing of all views after a change; it is then up to you to manually trigger updates when appropriate (this is useful when remotely using Klusters over a slow network connection)

Settings->Configure Shortcuts

Lets you configure the shortcuts

Settings->Configure klusters

Lets you configure a number of default behaviors
