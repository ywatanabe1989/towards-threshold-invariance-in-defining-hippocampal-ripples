2.10. Saving the Cluster File

To save a cluster file, select File->Save, or File->Save As if you wish to save the data in a different location on the disk.

By the time you are done editing your data, it is highly improbable that the remaining clusters are numbered in sequence. This can be corrected, by selecting File->Renumber and Save. Before the data is saved, the clusters will be renumbered starting from 2.
