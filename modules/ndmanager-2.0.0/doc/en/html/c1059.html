Chapter 5. Menu Entries

5.1. The File Menu

File->Open (Ctrl-O)

Opens data files

File->Open Recent

Opens data files selected from a list of recently edited data files

File->Import (Ctrl-I)

(not yet implemented)

File->Save (Ctrl-S)

Saves the cluster file

File->Save As

Saves the cluster file in a new location

File->Renumber and Save (Ctrl-Shift-S)

Renumbers the clusters in sequence and saves the cluster file

File->Print (Ctrl-P)

Prints the each view of the current display in a separate page.

File->Close

Closes the data files (will prompt for saving if necessary)

File->Quit (Ctrl-Q)

Quits Klusters (will prompt for saving if necessary)
