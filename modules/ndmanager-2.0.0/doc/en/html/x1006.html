7.4. The Help Menu

Help->ndmanager Handbook

Opens this user manual in the Help Center.

Help->What's this?

Activates the context help tool.

Help->Report Bug

Helps you compose an e-mail message to report a bug to the author of NDManager.

Help->About ndmanager

Shows information about the authors of and contributors to NDManager.

Help->About KDE

Shows information about the authors of and contributors to KDE.
