5.4. The Tools Menu

Tools->Zoom (Z)

Allows you to zoom on a point (left click) or area (click and drag) in most views; hold Shift to zoom out (except in the Trace View), or double-click to reset the zoom level to the initial state

Tools->New Cluster (N)

Allows you to draw a polygon to define a new cluster (click the left button to add a point to the polygon, the right button to remove the last point, or the middle button to automatically close the polygon)

Tools->Split Clusters (S)

Allows you to draw a polygon to define one new cluster for each existing cluster intersecting with the polygon

Tools->Delete Artefact Spikes (A)

Allows you to draw a polygon to select groups of spikes to move to the artefacts cluster (cluster 0)

Tools->Delete Noisy Spikes (N)

Allows you to draw a polygon to select groups of spikes to move to the noise cluster (cluster 1)

Tools->Select Time (W)

Allows you to select in the Cluster view the starting time use in the Trace view
