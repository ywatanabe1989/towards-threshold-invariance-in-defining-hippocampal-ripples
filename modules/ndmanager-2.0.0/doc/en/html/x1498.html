5.8. The Displays Menu

Displays->New Cluster Display

Creates a new Cluster Display

Displays->New Waveform Display

Creates a new Waveform Display

Displays->New Correlation Display

Creates a new Correlation Display

Displays->New Overview Display

Creates a new Overview Display

Displays->New Grouping Assistant Display

Creates a new Grouping Assistant Display

Displays->New Trace Display

Creates a new Trace Display

Displays->Rename Active Display (Ctrl-R)

Brings a dialog enabling the user to enter a new label for the currently active display.

Displays->Close Active Display (Ctrl-W)

Closes the currently active display; when there is only one display, closes all data files as well (will prompt for saving if necessary)
