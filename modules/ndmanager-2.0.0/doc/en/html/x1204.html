5.3. The Actions Menu

Actions->Delete Artifact Cluster(s) (Shift-DEL)

Deletes all currently selected clusters and moves all their spikes to the artefacts cluster (cluster 0)

Actions->Delete Noisy Cluster(s) (DEL)

Deletes all currently selected clusters and moves all their spikes to the noise cluster (cluster 1)

Actions->Group Clusters (G)

Groups all currently selected clusters

Actions->Update Display

Updates all currently visible views (delayed update mode)

Actions->Renumber Clusters (R)

Renumbers all clusters in sequence

Actions->Update Error Matrix (U)

Updates the error matrix

Actions->Recluster (Shift-R)

Launchs an external automatic cluster cutting program

Actions->Abort Reclustering

Kills the external automatic cluster cutting program currently running
