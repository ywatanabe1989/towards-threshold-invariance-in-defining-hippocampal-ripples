Chapter 3. Creation and Modification of Parameter Files

3.1. Starting a session

NDManager has two modes, a default mode and an Expert Mode. In the default mode, only the most commonly used features are available. The Expert Mode allows you to modify advanced features. To switch to the Expert Mode, select Settings->Expert Mode (selecting this menu item a second time will toggle its status and set the default mode back).

The following sections will describe all the available features (common features and advance features).

3.1.1. Creating a Parameter File

The creation of a new file is based on an existing file. This file can be either the default file or any existing file you wish.

If you select File->New. The default file will be used. NDManager comes with a default file containing a set of default values. This file can be overloaded. To set a current file as the default file, simply select File->Save as Default. Thereafter, when creating a new file, the default file you have chosen will be used.

To create a new file based on an existing file (different from the default file), simply select File->Import File as Model.... When saving the file you will be prompt for a new name.

3.1.2. Modifying a Parameter File

You can modify an existing parameter file at any time. To load an existing file select File->Open or File->Open Recent.

The different sections of the parameter file are presented on separated pages of a tree view. Each of those sections are explained in the following chapters.
