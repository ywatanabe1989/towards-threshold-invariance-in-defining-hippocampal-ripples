5.6. The Correlations Menu

Correlations->Scale by Asymptote (Shift-A)

Scales each correlogram by its asymptotic value

Correlations->Scale by Maximum (Shift-M)

Scales each correlogram by its maximum value

Correlations->Uniform Scale (Shift-U)

Scales all correlograms by the same value (determined by the default gain)

Correlations->Asymptote Line (L)

Shows or hides the asymptotic lines and average firing rates

Correlations->Increase Amplitude (Shift-I)

Increases the vertical scale

Correlations->Decrease Amplitude (Shift-D)

Decreases the vertical scale
